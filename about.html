Project Overview

What is this?

The SignDetect AI project aims to provide a fast and accessible tool for translating American Sign Language (ASL) gestures into text in real-time. By leveraging modern computer vision techniques, we strive to make communication more inclusive and intuitive. It can accurately identify 6 common sign language gestures: Hello, Yes, No, Thanks, Sorry, and Please.

Why is this important?

According to the World Health Organization, over 430 million people worldwide have hearing loss. Sign language is essential for communication in the deaf community, but most hearing individuals don't understand it. This technology bridges that gap, making communication more accessible and inclusive for everyone.

Technical Highlights

  • Python: The primary programming language for all backend logic and ML tasks.
  • OpenCV (`opencv-python`): Used for camera capture, image manipulation, and displaying output (in the original desktop version).
  • MediaPipe (`mediapipe`): The powerful framework for high-fidelity hand and landmark detection.
  • Scikit-learn (`scikit-learn`): The foundational library for machine learning, used here to implement the Random Forest Classifier.
  • NumPy: Essential for efficient array and data manipulation during dataset creation and inference.
