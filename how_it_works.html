The Detection Pipeline

This project follows a standard Machine Learning pipeline involving data collection, preprocessing, training, and real-time inference. Your Python scripts manage each phase seamlessly.

1. Data Collection

This script is used to capture multiple image samples for each sign (class) using the webcam. Images are stored in the `./data` directory, structured by class ID (0, 1, 2, etc.).

2. Dataset Creation

This is the preprocessing step. It iterates through the collected images, uses MediaPipe Hands to detect and extract the 21 hand landmarks for each successful detection. The normalized coordinates (features) and their corresponding sign label are saved into the "data.pickle"file for training.

3. Model Training

The features from `data.pickle` are used to train a powerful classification model. We use the Random Forest Classifier (`sklearn.ensemble`) for its accuracy and robustness. The trained model is then serialized and saved as "model.p" file

4. Real-Time Inference

This is the core detection script. It loads the `model.p`, accesses the webcam, and performs the following in real-time on every frame:

  1. Capture the video frame.
  2. Process the frame with MediaPipe to get hand landmarks.
  3. Normalize the landmark coordinates (same process as `create_dataset.py`).
  4. Pass the normalized features to the loaded Random Forest model for classification.
  5. Display the detected hand sign (e.g., 'HELLO', 'YES') on the screen.
